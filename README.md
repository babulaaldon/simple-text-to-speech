# Text-to-Speech PDF Reader

A Python application that reads the contents of a PDF file and converts it to speech using `pyttsx3`.

## Features

- Reads PDF files.
- Converts text from PDF to speech.
- Simple and easy-to-use interface.

## Getting Started

### Prerequisites

- Python 3.x
- `pyttsx3` library
- `PyPDF2` library
